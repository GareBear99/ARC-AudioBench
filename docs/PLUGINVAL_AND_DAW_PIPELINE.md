# pluginval and DAW Pipeline

ARC-AudioBench does not replace pluginval. It stores pluginval and DAW outputs as evidence alongside rendered audio analysis.

Suggested matrix: Reaper, Ableton, Logic, FL Studio, Bitwig, Cubase, Studio One; VST3/AU where applicable; 44.1/48/96 kHz; 64/256/1024 buffer sizes.
