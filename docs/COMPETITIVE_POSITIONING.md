# Competitive Positioning

ARC-AudioBench is not a replacement for pluginval, DAW testing, or formal DSP analysis tools. It is the local-first evidence layer that ties them together.

## Compared with pluginval

pluginval stress-tests plugin binaries. ARC-AudioBench stores benchmark signals, rendered WAV analysis, JSON results, reports, and release evidence.

## Compared with generic audio scripts

ARC-AudioBench is packaged as a public product with schemas, docs, examples, issue templates, citations, and ecosystem integrations.

## Compared with research notebooks

ARC-AudioBench is repo-native and release-oriented, so it can support open-source plugin release workflows as well as reproducible research.
