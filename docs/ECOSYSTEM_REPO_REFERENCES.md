# Ecosystem Repo References

ARC-AudioBench is the benchmark/evidence layer for the GareBear99 / TizWildin audio-tech ecosystem. It is not a monorepo clone of these projects; it references them as benchmark targets, governance models, SEO anchors, research bridges, and public-routing surfaces.

## How these repos are used

- **Product benchmark targets:** FreeEQ8, FreeVox8, Voxel Audio, and Instrudio provide concrete audio/plugin/Web Audio systems that ARC-AudioBench can validate.

- **Governance references:** ARC-Neuron LLMBuilder, ARC-Core, Arc-RAR, and OmniBinary Runtime define the evidence, release-gate, receipt, archive, and restore language used by ARC-AudioBench.

- **SEO and public discovery:** Awesome Audio Plugins & Dev, Awesome Audio Lists, Awesome Music Platforms, Awesome Python Audio Science, and the TizWildin HUB router make ARC-AudioBench part of a searchable public audio ecosystem.

- **Creator-resource routing:** TizWildin Release Vault and the HUB connect benchmark evidence to deconstructed loops, sample packs, visualizers, music releases, and creator tools.

## Required public launch cross-links

When ARC-AudioBench is published, link it from:

- `FreeEQ8` and `FreeVox8` as the official validation/evidence suite.

- `TizWildinEntertainmentHUB` under the public Lists / Tools router.
