# FreeEQ8 Benchmark Suite

Recommended FreeEQ8 tests:

1. Bypass null render using sine, sweep, and impulse.
2. Low-pass / high-pass sweep sanity render.
3. Bell boost/cut test at 100 Hz, 1 kHz, and 10 kHz.
4. M/S routing test using stereo source files.
5. Preset recall test.
6. Automation ramp test.
7. Analyzer/UI stress observation.
8. pluginval strictness 10.

Outputs should be saved under `results/freeeq8/`.
